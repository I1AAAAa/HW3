Understanding the 8 Cognitive Functions

Each Jungian cognitive function represents a unique way of perceiving or judging the world. Below are simplified explanations with example behaviors.

Visual Overview

INTJ — The Strategic Thinker

Dominant Function: Introverted Intuition (Ni)
Auxiliary Function: Extraverted Thinking (Te)

  • Visionary — Sees patterns and long-term outcomes.
  • Independent — Prefers working alone to refine ideas.
  • Analytical — Focuses on structure and efficiency.

ENFP — The Inspired Creator

Dominant Function: Extraverted Intuition (Ne)
Auxiliary Function: Introverted Feeling (Fi)

  • Curious — Loves exploring possibilities.
  • Empathetic — Deeply values authenticity.
  • Energetic — Motivates others with enthusiasm.

ISTP — The Analytical Problem-Solver

Dominant Function: Introverted Thinking (Ti)
Auxiliary Function: Extraverted Sensing (Se)

  1. Logical Decision Making — Prioritizes efficiency.
  2. Hands-On Learning — Learns through real experiences.
  3. Adaptable — Responds quickly to changing environments.

Summary of All Functions

Below is a simple reference list of all eight Jungian functions:

  • Ni — Introverted Intuition
  • Ne — Extraverted Intuition
  • Ti — Introverted Thinking
  • Te — Extraverted Thinking
  • Fi — Introverted Feeling
  • Fe — Extraverted Feeling
  • Si — Introverted Sensing
  • Se — Extraverted Sensing
